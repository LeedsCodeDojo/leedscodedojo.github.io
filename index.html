Leeds Code Dojo

Less talk, more code

What?

Leeds Code Dojo is a group dedicated to programming. We focus on practical coding sessions, where you can learn by doing.

Want to have some fun coding in a relaxed and friendly environment? Interested in self-improvement through deliberate practice? Want to see what you can learn from fellow programmers, and teach them what you know? Then Leeds Code Dojo is for you!


From newbie Node coders to hardcore Haskell hackers, and all the levels of C#, Java, Ruby, Python or F# inbetween, all languages and abilities are welcome.

When?

Next: The Pipe Packing Challenge - Thursday 6th November

Peter Piper Packed a Perfect Pack of Pristine Pipes. And so will you, after taking part in the Pipe Packing Challenge!

In a coding session taking inspiration* from the last leeds sharp session, this november we'll be writing algorithms to try and solve a simplified version of the bin packing problem.

We'll be working in pairs or small teams as usual. Unit Testing/TDD is recommended but optional for this session, which should please those of you who are too clever to need tests (*cough* Matt *cough*). The team with the most optimum solution by the end of the night will win.. A slice of (cold) pizza! And of course the admiration of all involved.

Doors are open from 18:00, and the night will kick off at 18:30 with food and chit-chat. We'll get started on the coding around 19:00, starting with an explanation of the problem and how we'd like to you solve it.

* Took inspiration, blatantly stole.. It's all coding fun, and that's the main thing, right?!

Please register on meetup if you'd like to attend.

Previous: Build you a Skynet - Wednesday 1st October

We worked in twos and threes to write Machine Learning algorithms based on the k-nearest neighbours algorithm. The group did well, with almost everyone getting a working solution, and some teams getting to over 90% accuracy with their digit classification!

As usual, there were several languages in use - F#, Objective C, Scala, C#, and Python, and again a few people got involved with a new language, so well done to them.


Some of the solutions are up on github - go check them out!
