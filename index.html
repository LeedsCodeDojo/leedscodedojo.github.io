Leeds Code Dojo

Less Talk More Code

Leeds Code Dojo is a group dedicated to programming. We focus on practical coding sessions, where you can learn by doing.

Want to have some fun coding in a relaxed and friendly environment? Interested in self-improvement through deliberate practice? Want to see what you can learn from fellow programmers, and teach them what you know? Then Leeds Code Dojo is for you!
From newbie Node coders to hardcore Haskell hackers, and all the levels of C#, Java, Ruby, Python or F# inbetween, all languages and abilities are welcome.

Refuctoring - Wednesday 3rd December

In this session, you are going to create some of the best-written, cleanest code you've ever seen. Then, one small step at a time, you're going to utterly destroy it!

You may be familiar with Refactoring, by which you try to eliminate 'smells' in your code and make it gradually better. Refuctoring is the exact opposite, whereby you try to identify and remove 'smiles' in your code in the hope of making it almost impossible to read. As Jason Gorman, who coined the phrase, puts it:

"Refuctoring is the process of taking a well-designed piece of code and, through a series of small, reversible changes, making it completely unmaintainable by anybody except yourself. Comprehensive regression testing guarantees that nobody will be any the wiser."

The hope is that in studying the ways in which code can be made worse, you will learn to appreciate and improve your ability to make it better. And even if you don't, you're guaranteed to have some fun! There may even be a prize for the pair with the worst-looking code.

Doors are open from 18:00, and the night will kick off at 18:30 with food and socialising. We'll get started on the coding around 19:00, coding up a simple solution in pairs using TDD before the real fun begins...

You'll be using whatever languages you have available - so bring your laptop, if you have one, with your language of choice installed including some way of running unit tests. We are very limited in power sockets, so make sure your battery is fully charged!

This night is suitable for any level of experience, but you do need to have some knowledge of programming.

We'll finish up around 21:00, and more than likely head to the pub afterwards.

Please register on meetup if you'd like to attend.

Hope to see you there!
