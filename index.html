Leeds Code Dojo is a group dedicated to programming. We focus on practical coding sessions, where you can learn by doing.

Want to have some fun coding in a relaxed and friendly environment? Interested in self-improvement through deliberate practice? Want to see what you can learn from fellow programmers, and teach them what you know? Then Leeds Code Dojo is for you!
From newbie Node coders to hardcore Haskell hackers, and all the levels of C#, Java, Ruby, Python or F# inbetween, all languages and abilities are welcome.

Try a Language: Elixir - Wednesday 2nd December

In the second of our 'try a language' series, Phil Helliwell will be showing us around another functional language, this time in the form of Elixir.

According to the website, "Elixir is a dynamic, functional language designed for building scalable and maintainable applications. Elixir leverages the Erlang VM, known for running low-latency, distributed and fault-tolerant systems, while also being successfully used in web development and the embedded software domain."

Or as Phil puts it, "imagine what happens when Ruby developers get hold of Erlang; the result is Elixir". Sounds good to me!

At least half of you will need to install Elixir, so check out the installation instructions and get that on. You'll also need a telnet client, which I think comes as standard on Unix-like systems - for Windows, try PuTTY.

Doors are open from 18:00 for a bit of light socialising, pizza will arrive around 18:15, and we'll get started on the coding at 18:45. We have a very limited number of power sockets, so make sure your come fully charged!

This night is suitable for any level of experience, but you do need to have some knowledge of programming.

We'll finish up around 21:00, and head to Mr Foleys afterwards for a refreshing ale or two.

Please register on meetup if you'd like to attend.

Hope to see you there!
