Leeds Code Dojo

Less Talk More Code

Leeds Code Dojo is a group dedicated to programming. We focus on practical coding sessions, where you can learn by doing.

Want to have some fun coding in a relaxed and friendly environment? Interested in self-improvement through deliberate practice? Want to see what you can learn from fellow programmers, and teach them what you know? Then Leeds Code Dojo is for you!
From newbie Node coders to hardcore Haskell hackers, and all the levels of C#, Java, Ruby, Python or F# inbetween, all languages and abilities are welcome.
Come and join our Meetup group to get alerts about upcoming events and join in the coding chat!

Checkout Kata - The Redoux! - Wednesday 4th February

Following Kiran's suggestion from the January dojo, this month we're once again going to attempt the Checkout Kata, whereby you implement the core algorithm for a supermarket checkout.

The idea of tackling the same problem again is that it allows you to try something new - a different approach to the problem, a new language, a different coding partner, and so on.

To be clear, we're starting again from scratch, not continuing from where we left off, so it's not a problem if you didn't come in January. If you'd like to see some of January's solutions, have a look at the github repo.

Doors are open from 18:00 with a bit of light socialising, pizza will arrive around 18:30, and we'll get started on the coding at 19:00.

You'll be using whatever languages you have available - so bring your laptop, if you have one, with your language of choice installed including some way of running unit tests. We have a very limited number of power sockets, so make sure your come fully charged!

This night is suitable for any level of experience, but you do need to have some knowledge of programming.

We'll finish up around 21:00, and more than likely head to the pub afterwards.

Please register on meetup if you'd like to attend.

Hope to see you there!
